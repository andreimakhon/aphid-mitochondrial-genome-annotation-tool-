Для работы программы необходимо установить:
Python v3.5+
Biopython v1.78+
Blastp 2.9.0+
MUSCLE v3.8.31+

Для запуска программы в командной строке необходимо перейти в папку программы и написать: 
    Python aphid_mito_annotation.py -i <путь к файлу для аннотации> -o <путь к файлу с результатами в формате .xlsx>
